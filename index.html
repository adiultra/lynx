Welcome to Lynx

The definitive Linux Blog

Linux is a Free and Open-Source Operating System for people like us and everyone else. It runs on almost all hardware be it a super computer or minuscule Raspberry pi. It provides unlimited power of customization, coupled with amazing speed and a very secure architecture. It is all you need for your home needs or spesialized work.

Although Linux by default contains many enhancements over other Operating systems, It is often thought to be hard and very confusing. It is a baseless fact. Anyone who has ever tried Linux can immediately tell you that it is one of the best OS around. And for a good reason. If you say you never used linux, You are a very bad lier. Because linux is the main core behind Android, the most popular mobile OS.

This blog aims to provide Tips Tricks and help to all those who want to step into linux and find it confusing. You can start with the beginner's section and discover the fun that awaits you.
